## 🚀 O que desenvolvi:

📦 **Sistema de Cadastro de Pessoas (POO)**
Um gerenciador completo via console que aplica conceitos fundamentais de arquitetura e manipulação de dados em Java.
* **Arquitetura de Software**: Organização do código em camadas (Model, Service e View) para garantir a separação de responsabilidades.
* **Ordenação e Busca**: Uso de `Comparator` com `toLowerCase()` para listagem alfabética precisa e filtros flexíveis com `equalsIgnoreCase`.
* **Robustez e UX**: Tratamento de exceções com `try-catch` para garantir a continuidade do programa e gestão de buffer do `Scanner` (`sc.nextLine()`) para uma experiência de usuário fluida.
* **Tecnologias**: Java, Collections (ArrayList), e Programação Orientada a Objetos.

# Repositório de Estudos: Backend Java

Este repositório documenta minha evolução técnica e a aplicação prática de conceitos aprendidos durante a graduação em Análise e Desenvolvimento de Sistemas. O foco atual é a transição da lógica de programação estruturada para arquiteturas robustas utilizando **Java** e **Programação Orientada a Objetos (POO)**.

## Projetos em Destaque

### Sistema de Cadastro de Pessoas (POO)
Gerenciador de dados via console que implementa padrões de organização de código e manipulação eficiente de coleções.

*   **Arquitetura**: Organização em camadas (Model, Service e View) para separação de responsabilidades e facilidade de manutenção.
*   **Manipulação de Dados**: Implementação de `Comparator` com `toLowerCase()` para ordenação alfabética e buscas flexíveis com `equalsIgnoreCase`.
*   **Tratamento de Erros**: Uso de blocos `try-catch` para garantir a resiliência do sistema e gestão de buffer do `Scanner` para uma interação estável.
*   **Tecnologias**: Java SE, Collections API (ArrayList), POO.
