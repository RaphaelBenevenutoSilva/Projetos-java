## 🚀 O que desenvolvi:

📦 **Sistema de Cadastro de Pessoas (POO)**
Um gerenciador completo via console que aplica conceitos avançados de arquitetura e manipulação de dados.
* **Destaque Técnico**: Implementação de arquitetura dividida (Model, Service e View) para separação de responsabilidades.
* **Ordenação e Busca**: Uso de `Comparator` com `toLowerCase()` para listagem alfabética precisa e filtros flexíveis com `equalsIgnoreCase`.
* **Robustez**: Tratamento de exceções com `try-catch` para garantir a continuidade do programa contra erros de input.

🗳️ **Simulador de Paredão (BBB)**
Sistema que automatiza a lógica de votação baseada em pesos e percentuais reais.
* **Destaque**: Uso de média ponderada para o resultado final e organização de nomes em Arrays para facilitar o cálculo dentro de laços `for`.
